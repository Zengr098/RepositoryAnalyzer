About

Repository Analyzer è un plug-in per Google Chrome che permette di analizzare una qualsiasi repository GitHub pubblica. L'input che deve essere dato in pasto al plug-in è l'URL della repository in questione.

Il nostro obiettivo è quello di quantificare l'effettivo contributo degli sviluppatori che partecipano ad un progetto open source.
